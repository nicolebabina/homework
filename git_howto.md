# Подсказка по GIT

Создание репозитория: 
```sh
git init
```

Отображение статуса:
```sh
git status
```

Добавить:
```sh
git add
```

Сохранить + комментарий:
```sh 
git commit -m "Message"
```

Просмотр всех действий: 
```sh
git log
```

Просмотр всех действий (короткий вариант):
```sh
git log --oneline
```

Вoзвращение к ветке HEAD:
```sh
git checkout
```
Перемещение указателя HEAD:
```sh
git reset
```

Разница между сохраненнымим и нет ветками:
```sh
git diff
```

Просмотр всех веток
```sh
git branch
```

Переход на другую ветку
```sh
git checkout branch_name
```

Удаление ветки
```sh
git branch -d branch_name
```

Совмещение двух веток
```sh
git merge branch_name
```

Изменение назвиния ветки
```sh
git branch new_branch_name
```

История в виде древа
```sh
git log --graph
```

Очистить terminal
```sh
clear
```
