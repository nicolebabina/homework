# Подсказка по GIT

Создание репозитория: 
```sh
git init
```

Отображение статуса:
```sh
git status
```

Добавить:
```sh
git add
```

Сохранить + комментарий:
```sh 
git commit -m "Message"
```

Просмотр всех действий: 
```sh
git log
```

Просмотр всех действий (короткий вариант):
```sh
git log --oneline
```

Вщзвращение к ветке HEAD:
```sh
git checkout
```
